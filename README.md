# Mock API Directory

This directory contains the mock server implementation for the dashboard.
Created by DS Team 3 immediately after contract session (Days 2-3).

## Purpose

- Provide hardcoded sample responses conforming to contracts.
- Allow dashboard development to proceed in parallel with model development.

## Usage

The dashboard is configured to point to this mock server initially via `.env` variables `CHURN_API_URL` and `REC_API_URL`.
